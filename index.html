La Combustion

Réaction entre un combustible et un comburant (O₂)

📓 Prends des notes sur ton cahier de brouillon !

🔥 Définition

La combustion est une réaction chimique exothermique entre :

  • Un combustible : hydrocarbure (carbone C, hydrogène H, parfois oxygène O et soufre S)
  • Un comburant : le dioxygène O₂ de l'air
⚠️ À retenir

Combustion complète → produits : CO₂ + H₂O (+ SO₂ si soufre)

Combustion incomplète → produits : CO₂ + CO + H₂O (manque d'O₂)

⚖️ Écrire et équilibrer une combustion

Méthode en 4 étapes

Étape 1 – Écrire les réactifs et produits

Combustible + O₂ → CO₂ + H₂O

Étape 2 – Équilibrer C

Coefficient devant CO₂ = nombre d'atomes de C dans le combustible

Étape 3 – Équilibrer H

Coefficient devant H₂O = (nombre d'atomes H) ÷ 2

Étape 4 – Équilibrer O

Compter tous les O à droite → coefficient devant O₂

Exemple : combustion du méthane CH₄

CH₄ + 2 O₂ → 1 CO₂ + 2 H₂O

C : 1→1 ✓  |  H : 4→4 ✓  |  O : 4→4 ✓

Exemple : combustion du propane C₃H₈

C₃H₈ + 5 O₂ → 3 CO₂ + 4 H₂O

C : 3→3 ✓  |  H : 8→8 ✓  |  O : 10→10 ✓

Exemple : combustion du butane C₄H₁₀

2 C₄H₁₀ + 13 O₂ → 8 CO₂ + 10 H₂O

📐 Formule générale des alcanes

CnH2n+2 + ( (3n+1)/2 ) O₂ → n CO₂ + (n+1) H₂O
🧠 Astuce mnémotechnique

C avant tout, H ensuite, O en dernier — toujours dans cet ordre pour équilibrer !

🌡️ Enthalpie de combustion

La réaction libère de l'énergie : ΔH < 0 (réaction exothermique)

À savoir

L'enthalpie de combustion est toujours négative pour un combustible. Plus elle est grande (en valeur absolue), plus le combustible est énergétique.

Exemple : ΔH°comb(CH₄) = −890 kJ/mol

Tableau d'avancement

Suivre l'évolution d'une réaction chimique

📊 Principe

Le tableau d'avancement permet de suivre l'évolution des quantités de matière des espèces chimiques au cours d'une réaction.

Variable d'avancement x

x est en moles (mol). Elle augmente de 0 jusqu'à xmax (fin de réaction).

Pour une réaction : a A + b B → c C + d D

État Avancement n(A) n(B) n(C) n(D)
Initial x = 0 n₀(A) n₀(B) 0 0
En cours x n₀(A) − ax n₀(B) − bx cx dx
Final xmax n₀(A)−axmax n₀(B)−bxmax cxmax dxmax

Les coefficients a, b, c, d sont les coefficients stœchiométriques de l'équation.

🔍 Réactif limitant

Le réactif limitant est celui qui est entièrement consommé en premier. Il détermine xmax.

Méthode

Pour chaque réactif, calculer : n₀(réactif) / coefficient stochio

Le réactif limitant est celui qui donne la valeur minimale.

xmax = min( n₀(A)/a , n₀(B)/b )

Exemple concret

Réaction : N₂ + 3 H₂ → 2 NH₃

On mélange 2 mol de N₂ et 4 mol de H₂.

N₂ : 2/1 = 2 mol H₂ : 4/3 = 1,33 mol ← minimum → H₂ est le réactif limitant xmax = 1,33 mol

🔢 Exemple complet — Combustion du propane

Réaction : C₃H₈ + 5 O₂ → 3 CO₂ + 4 H₂O

Données : n₀(C₃H₈) = 0,5 mol  |  n₀(O₂) = 3 mol

Réactif limitant ?

C₃H₈ : 0,5 / 1 = 0,5 O₂ : 3 / 5 = 0,6 → C₃H₈ est le réactif limitant → xmax = 0,5 mol
État x (mol) n(C₃H₈) n(O₂) n(CO₂) n(H₂O)
Initial 0 0,5 3 0 0
En cours x 0,5 − x 3 − 5x 3x 4x
Final 0,5 0 0,5 1,5 2

💡 Taux d'avancement

τ = xmax / xmax théo

Si τ = 1 → réaction totale. Si τ < 1 → réaction incomplète (équilibre chimique).

Oxydoréduction

Transfert d'électrons entre oxydants et réducteurs

⚡ Définitions fondamentales

Oxydation

Une espèce perd des électrons → elle est le réducteur (Réd₁). Elle est oxydée.

Réduction

Une espèce gagne des électrons → elle est l'oxydant (Ox₂). Elle est réduite.

🧠 Moyen mémo

OILRIG : Oxidation Is Loss, Reduction Is Gain (d'électrons)

Ou en français : LEO le lion dit GER – Lose Electrons Oxidation / Gain Electrons Reduction

🔗 Couples oxydant/réducteur

Un couple redox s'écrit : Ox / Réd

Ox + n e⁻ ⇌ Réd

Exemples de couples importants :

Cu²⁺ / Cu → Cu²⁺ + 2e⁻ ⇌ Cu Fe³⁺ / Fe²⁺ → Fe³⁺ + e⁻ ⇌ Fe²⁺ Zn²⁺ / Zn → Zn²⁺ + 2e⁻ ⇌ Zn H⁺ / H₂ → 2H⁺ + 2e⁻ ⇌ H₂ MnO₄⁻/Mn²⁺ → MnO₄⁻ + 8H⁺ + 5e⁻ ⇌ Mn²⁺ + 4H₂O

⚖️ Nombre d'oxydation (n.o.)

Règles de base

• Élément simple (Na, Fe, Cu…) : n.o. = 0

• Ion monoatomique (Fe²⁺) : n.o. = charge de l'ion

• H dans molécule : n.o. = +I (sauf hydrures métalliques : −I)

• O dans molécule : n.o. = −II (sauf H₂O₂ : −I ; OF₂ : +II)

• Somme des n.o. = charge totale de l'espèce

Exemple : n.o. du Mn dans MnO₄⁻

n.o.(Mn) + 4×(−II) = −I n.o.(Mn) − 8 = −1 n.o.(Mn) = +VII

🔄 Règle du gamma (γ)

Pour combiner les deux demi-équations et que les électrons s'annulent :

Principe

Multiplier chaque demi-équation par un coefficient pour que le nombre d'électrons échangés soit identique dans les deux demi-réactions.

Ox₁ + n₁ e⁻ → Réd₁ (×n₂) Réd₂ → Ox₂ + n₂ e⁻ (×n₁) ──────────────────────── Ox₁×n₂ + Réd₂×n₁ → Réd₁×n₂ + Ox₂×n₁

Demi-équations électroniques

Équilibrer les demi-réactions d'oxydation et de réduction

⚗️ Méthode d'équilibrage

On suit 5 étapes dans l'ordre :

① Écrire les espèces Ox et Réd

Repérer l'élément qui change de nombre d'oxydation.

② Équilibrer l'élément principal

Ajuster les coefficients pour équilibrer l'atome qui change de n.o.

③ Équilibrer O avec H₂O

Ajouter des molécules H₂O du côté qui manque d'oxygène.

④ Équilibrer H avec H⁺

Ajouter des ions H⁺ du côté qui manque d'hydrogène.

⑤ Équilibrer les charges avec e⁻

Ajouter des électrons e⁻ pour équilibrer les charges totales. Les e⁻ sont à gauche si réduction, à droite si oxydation.

✏️ Exemples détaillés

Exemple 1 : Couple Cu²⁺ / Cu (simple)

Cu²⁺ + 2e⁻
→
Cu

Cu passe de n.o. +II à 0 → gain de 2e⁻ → réduction

Exemple 2 : Couple MnO₄⁻ / Mn²⁺ (milieu acide)

① MnO₄⁻ → Mn²⁺   ② Mn équilibré   ③ ajouter 4 H₂O   ④ ajouter 8 H⁺   ⑤ charges :

Charges à gauche : −1 + 8(+1) = +7 Charges à droite : +2 Différence : 7 − 2 = 5 e⁻ à gauche
MnO₄⁻ + 8H⁺ + 5e⁻
→
Mn²⁺ + 4H₂O

Exemple 3 : Couple Cr₂O₇²⁻ / Cr³⁺ (milieu acide)

① Cr₂O₇²⁻ → 2 Cr³⁺ (équilibrer Cr : ×2) ③ 7 H₂O côté droit ④ 14 H⁺ côté gauche ⑤ Charges gauche : −2+14 = +12 / droite : 6 12 − 6 = 6e⁻ à gauche
Cr₂O₇²⁻ + 14H⁺ + 6e⁻
→
2Cr³⁺ + 7H₂O

Exemple 4 : Oxydation du fer Fe²⁺ → Fe³⁺

Fe²⁺
→
Fe³⁺ + e⁻

Fe passe de +II à +III → perte de 1e⁻ → oxydation

🔗 Combiner les demi-équations

Exemple : réaction entre MnO₄⁻ et Fe²⁺

Réduction : MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O (×1) Oxydation : Fe²⁺ → Fe³⁺ + e⁻ (×5) ───────────────────────────────────────────── MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺

Les 5e⁻ s'annulent → équation bilan correcte ✓

✅ Vérification obligatoire

Vérifier que chaque atome et chaque charge est équilibré des deux côtés !

Tableau d'avancement – réaction acido-basique Facile

Q2 : HCl : 0,4/1 = 0,4  |  NaOH : 0,3/1 = 0,3 → NaOH est le réactif limitant → xmax = 0,3 mol

Combustion + tableau d'avancement Moyen

On brûle 0,2 mol de propane C₃H₈ avec 0,8 mol de O₂.

Rappel : C₃H₈ + 5 O₂ → 3 CO₂ + 4 H₂O

Q1. Déterminer le réactif limitant.

Q2. Construire le tableau d'avancement complet.

Q3. Calculer les quantités de matière de CO₂ et H₂O formées.

Q4. La combustion est-elle complète ? Justifier.

✅ Correction

Q1 : C₃H₈ : 0,2/1 = 0,2  |  O₂ : 0,8/5 = 0,16 → O₂ est le réactif limitant → xmax = 0,16 mol

Q3 : n(CO₂) = 3 × 0,16 = 0,48 mol  |  n(H₂O) = 4 × 0,16 = 0,64 mol

Q4 : Non, car du propane reste (0,2 − 0,16 = 0,04 mol) → combustion incomplète (manque d'O₂).

Nombre d'oxydation & identification redox Moyen

Pour chaque réaction, identifier l'oxydant, le réducteur, et dire si chaque espèce est oxydée ou réduite.

Q1. Zn + 2 HCl → ZnCl₂ + H₂  —  Calculer le n.o. de Zn avant et après, et de H avant et après.

Q2. 2 Fe³⁺ + Sn²⁺ → 2 Fe²⁺ + Sn⁴⁺  —  Identifier oxydant et réducteur.

Q3. Cu + 2 Ag⁺ → Cu²⁺ + 2 Ag  —  Écrire les deux demi-équations électroniques.

✅ Correction

Q1 : Zn : 0 → +II (oxydation, Zn est le réducteur)  |  H : +I → 0 (réduction, H⁺ est l'oxydant)

Q2 : Fe³⁺ est l'oxydant (Fe³⁺ → Fe²⁺, gain d'e⁻)  |  Sn²⁺ est le réducteur (Sn²⁺ → Sn⁴⁺, perte d'e⁻)

Q3 : Réduction : Ag⁺ + e⁻ → Ag  |  Oxydation : Cu → Cu²⁺ + 2e⁻

Q2. Écrire et équilibrer la demi-équation d'oxydation du couple Fe³⁺ / Fe²⁺.

Q4. Quelle est la couleur de MnO₄⁻ ? Et de Mn²⁺ ? Comment peut-on savoir que la réaction est terminée ?

Exercice de synthèse complet Difficile

Q4. Construire le tableau d'avancement (les quantités sont en mmol).

Q5. Quel est le réactif limitant ? Calculer les quantités finales de Cr³⁺ et Fe³⁺.

✅ Correction

Q1 : Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O

Q2 : Fe²⁺ → Fe³⁺ + e⁻
